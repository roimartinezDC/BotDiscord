## BotDiscord
Un bot de Discord

                                                                        

Lista de comandos:                                                           
  !ping - devuelve una respuesta                                                    
  !embed - muestra un mensaje con una imagen                                                                
  /list - lista una serie de imagenes                                                        
  /get (nombre archivo) - muestra un mensaje *!embed* con el archivo disponible indicado
